# tableau-ga4-ecommerce-dashboard
Interactive ecommerce analytics dashboard built in Tableau using GA4 public dataset from BigQuery and SQL transformations.
# GA4 Ecommerce Analytics Dashboard

## Project Overview
This project analyzes ecommerce performance and user behavior using SQL (BigQuery) and Tableau Public.

## Key Metrics
- Sessions
- Purchases
- Revenue
- Conversion Rate
- Ecommerce Conversion Funnel
- Sessions by Device
- Sessions by Country
- Sessions by Landing Page
- Sessions by Operating System

## Tools
- SQL (BigQuery)
- Tableau Public
- Google Analytics 4

## Skills Demonstrated
- SQL
- BigQuery
- Tableau Public
- Google Analytics 4 (GA4)
- Data Analysis
- Data Visualization
- Web Analytics
- Funnel Analysis
- KPI Reporting
- Dashboard Development
